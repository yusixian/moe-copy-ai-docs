## What is MoeCopy AI?

MoeCopy AI is a lightweight browser extension built with the Plasmo framework that:

- **Smart Extraction**: Automatically identifies and extracts content from web pages
- **Multiple Modes**: Supports selector mode, reading mode, and hybrid extraction
- **AI Integration**: Works seamlessly with various AI models (OpenAI, Anthropic, etc.)
- **Customizable**: Configure custom selectors for specific websites
- **Privacy-Focused**: All data processing happens locally in your browser
