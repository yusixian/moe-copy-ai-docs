MoeCopy AI 是智能网页数据提取助手，能快速从网页中提取结构化数据。

## 安装扩展

### Chrome 网上应用店（推荐）

1. 访问 [Chrome 网上应用店](https://chromewebstore.google.com/detail/moe-copy-ai/dfmlcfckmfgabpgbaobgapdfmjiihnck)
2. 点击"添加至 Chrome" → 确认添加
3. 安装完成后工具栏出现 MoeCopy AI 图标

### Firefox 附加组件

1. 访问 [Firefox 附加组件商店](https://addons.mozilla.org/zh-CN/firefox/addon/moe-copy-ai/)
2. 点击"添加到 Firefox" → 确认添加

### 源码安装（开发者）

```bash
git clone https://github.com/yusixian/moe-copy-ai.git
cd moe-copy-ai
pnpm install
pnpm build
```

在 Chrome 中加载：`chrome://extensions/` → 开启开发者模式 → 加载已解压的扩展程序 → 选择 `build/chrome-mv3-prod`

## 基本使用

### 1. 打开目标网页

访问任何想提取数据的网页：新闻、博客、产品页面、文档等。

### 2. 点击扩展图标

点击工具栏中的 MoeCopy AI 图标，或点击页面右下角的悬浮按钮。

### 3. 自动提取数据

MoeCopy AI 会自动分析页面并提取：标题、作者、发布日期、正文内容、图片列表、元数据信息。

### 4. 复制或导出

- 点击"复制"按钮复制特定字段
- 使用"下载"导出为 JSON 或 Markdown
- 使用 AI 功能生成摘要或翻译

### 5. 使用侧边栏（高级功能）

<Badge>v0.2.0 新增</Badge>

右键扩展图标 → "在侧边栏打开"：

- **批量抓取**：选择页面区域，批量抓取多个链接中的文档
- **内容提取**：选择页面元素，提取为 HTML/Markdown/纯文本
- **设置**：配置 AI 和抓取选项

## 使用示例

以新闻文章为例：

1. 打开任意新闻文章页面
2. 点击扩展图标，MoeCopy AI 开始分析
3. 查看提取结果：标题、作者、发布时间、格式化内容、图片
4. （可选）切换到 AI 摘要标签页，输入"总结这篇文章的要点"生成摘要
