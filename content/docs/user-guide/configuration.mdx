## 访问配置页面

- 右键扩展图标 → 选项
- `chrome://extensions/` → MoeCopy AI 详情 → 扩展程序选项
- 扩展弹窗中点击设置图标

## 配置选项

### AI 设置

API 密钥存在本地，请妥善保管。不同模型能力和价格不同，注意 token 消耗。

#### API 配置

填入 API Key 与 Base URL，点击获取模型列表。

#### 提示词模板

支持变量替换：

```
{{content}} - 文章内容
{{title}} - 文章标题
{{url}} - 文章 URL
{{author}} - 作者
{{publishDate}} - 发布日期
{{cleanedContent}} - 清理后的内容
{{meta.xxx}} - 元数据字段
```

### 抓取设置

#### 抓取模式

- **选择器模式**：CSS 选择器精确提取
- **Readability 模式**：Mozilla Readability 算法
- **混合模式**：智能切换，优先选择器

#### 抓取时机

- **自动抓取**：页面加载完成后自动提取
- **手动抓取**：仅在点击悬浮球时提取

### 界面设置

#### 悬浮按钮

- 显示：在支持的页面显示悬浮球
- 隐藏：仅通过扩展图标访问

#### 主题设置

跟随系统 / 浅色模式 / 深色模式

### 选择器配置

#### 内容选择器

```css
/* 默认 */
main, #content, .content, .article-content

/* 自定义 */
.custom-article-body
#post-content
div[itemprop="articleBody"]
```

#### 标题选择器

```css
/* 默认 */
h1, meta[property="og:title"]

/* 自定义 */
.article-title
h1.post-title
```

#### 作者选择器

```css
/* 默认 */
meta[name="author"], [class*="author"]

/* 自定义 */
.author-name
span.by-author
```

#### 日期选择器

```css
/* 默认 */
meta[property="article:published_time"], time

/* 自定义 */
.publish-date
time[datetime]
```
