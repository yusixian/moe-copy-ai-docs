## 安装方式

### Chrome 网上应用店（推荐）

1. 访问 [Chrome 网上应用店](https://chromewebstore.google.com/detail/moe-copy-ai/dfmlcfckmfgabpgbaobgapdfmjiihnck)
2. 点击"添加至 Chrome"
3. 确认权限后完成安装
4. 建议置顶扩展图标

### Firefox 附加组件

1. 访问 [Firefox 附加组件商店](https://addons.mozilla.org/zh-CN/firefox/addon/moe-copy-ai/)
2. 点击"添加到 Firefox"
3. 确认权限后完成安装

### 开发者模式安装

适合开发者或想体验最新功能的用户。

#### 获取源代码

```bash
git clone https://github.com/yusixian/moe-copy-ai.git
cd moe-copy-ai
```

#### 安装依赖

```bash
pnpm install  # 推荐
# 或 npm install / yarn install
```

#### 构建扩展

```bash
# Chrome
pnpm dev    # 开发版本
pnpm build  # 生产版本

# Firefox (v0.2.1+)
pnpm dev:firefox    # 开发版本
pnpm build:firefox  # 生产版本
```

#### 加载到浏览器

**Chrome**:
1. 打开 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `build/chrome-mv3-dev` 或 `build/chrome-mv3-prod`

**Firefox** (v0.2.1+):
1. 打开 `about:debugging#/runtime/this-firefox`
2. 点击"临时载入附加组件"
3. 选择 `build/firefox-mv2-dev` 或 `build/firefox-mv2-prod` 目录下的 `manifest.json`

### 移动端安装（Android）

1. 安装 [Kiwi Browser](https://play.google.com/store/apps/details?id=com.kiwibrowser.browser)（或从 [GitHub Releases](https://github.com/kiwibrowser/src.next/releases) 下载）
2. 在 Kiwi Browser 中访问 Chrome 商店安装扩展

## 权限说明

### 必需权限

- **activeTab**: 访问当前标签页内容，用于提取网页数据
- **storage**: 保存用户设置和 AI 聊天历史
- **host_permissions**: 访问 `https://*/*` 用于内容提取

### 权限用途

- **网页内容访问**: 仅在用户主动触发时读取，不会后台收集数据，不访问密码或敏感表单
- **存储权限**: 保存选择器配置、AI API 设置、对话历史（本地存储）
- **网络权限**: 仅用于调用 AI API，不上传网页内容到第三方服务器

## 初始配置

### 首次启动

安装后建议进行配置：右键扩展图标 → 选项

### AI 配置（可选）

1. 在选项页面找到"AI 设置"
2. 填写 API 配置：API Key、API URL、模型
3. 保存设置

### 界面设置

- **抓取时机**: 自动或手动触发
- **调试面板**: 开发者可开启查看详细信息

> **提示**: v0.2.1 起，悬浮按钮功能已整合至侧边栏的"单页抓取"标签页。

## 更新和卸载

### 自动更新

从 Chrome 网上应用店安装的扩展会自动更新。

### 卸载扩展

访问 `chrome://extensions/` → 找到 MoeCopy AI → 移除

## 故障排除

### 扩展图标不显示

检查扩展是否已启用，点击拼图图标将 MoeCopy AI 固定到工具栏。

### 无法提取内容

1. 确保网页已完全加载
2. 检查是否是受保护页面（如银行网站）
3. 刷新页面后重试
