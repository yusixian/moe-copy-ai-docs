# 功能特性

MoeCopy AI 提供了一套完整的网页数据提取和处理功能，让您能够轻松地从任何网页中获取所需的结构化数据。

## 核心功能

### 🎯 智能内容提取

MoeCopy AI 使用多层级的智能选择器系统，能够准确识别和提取网页中的关键内容：

- **自动识别文章结构**：智能识别标题、作者、发布时间等元素
- **多级选择器回退**：从专用选择器到通用选择器的智能降级
- **自定义选择器支持**：为特定网站配置专属提取规则

### 📊 三种提取模式

根据不同场景选择最适合的提取方式：

1. **选择器模式（Selector）**
   - 基于 CSS 选择器的精确提取
   - 适合结构化良好的网站
   - 支持自定义选择器配置

2. **阅读模式（Readability）**
   - 使用 Mozilla Readability 算法
   - 智能识别正文内容
   - 自动过滤广告和无关元素

3. **混合模式（Hybrid）**
   - 结合两种模式的优势
   - 优先使用选择器，失败时回退到阅读模式
   - 提供最佳的兼容性

### 🤖 AI 智能处理

集成先进的 AI 功能，让数据处理更加智能：

- **内容摘要**：自动生成文章摘要和要点
- **智能翻译**：支持多语言翻译
- **自定义提示词**：根据需求定制 AI 处理方式
- **流式输出**：实时显示 AI 生成结果

### 🖼️ 图片管理

完善的图片提取和展示功能：

- **智能图片识别**：自动提取文章中的所有图片
- **懒加载支持**：识别各种延迟加载属性
- **图片预览**：网格化展示，支持放大查看
- **元数据保留**：保存图片的 alt、title 等信息

## 高级特性

### 📝 元数据提取

自动提取网页的丰富元数据：

- Open Graph 标签
- Twitter Card 信息
- Schema.org 结构化数据
- 自定义 meta 标签

### 🔧 开发者工具

为开发者和高级用户提供的专业功能：

- **选择器测试器**：实时测试 CSS 选择器效果
- **调试面板**：查看详细的提取过程和结果
- **日志系统**：可配置的日志级别，便于问题排查

### 💾 数据导出

灵活的数据导出选项：

- **JSON 格式**：完整的结构化数据
- **Markdown 格式**：便于阅读和编辑
- **纯文本**：简洁的文本内容
- **自定义模板**：使用模板引擎定制导出格式

### 🎨 用户界面

精心设计的用户体验：

- **响应式设计**：支持桌面和移动设备
- **深色模式**：自动适应系统主题
- **悬浮按钮**：快速访问，不干扰浏览
- **实时编辑**：直接编辑提取的内容
