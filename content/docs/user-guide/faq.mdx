## 安装问题

### Chrome 商店找不到扩展？

1. 某些地区可能无法访问，尝试使用[直接链接](https://chromewebstore.google.com/detail/moe-copy-ai/dfmlcfckmfgabpgbaobgapdfmjiihnck)
2. 搜索关键词：MoeCopy AI 或 moe copy
3. Firefox 用户访问 [Firefox 附加组件商店](https://addons.mozilla.org/zh-CN/firefox/addon/moe-copy-ai/)

### 开发者模式安装失败？

```bash
# 清理并重新安装
rm -rf node_modules pnpm-lock.yaml
pnpm install
pnpm build
```

确保 Node.js >= 16，加载扩展时选择 `build/chrome-mv3-prod` 目录。

### 移动端无法安装？

Chrome 移动版不支持扩展，使用 [Kiwi Browser](https://play.google.com/store/apps/details?id=com.kiwibrowser.browser)（Android）按桌面版步骤安装。

## 使用问题

### 为什么某些网页无法提取内容？

- **网页未加载完成**：等待加载完成或刷新
- **动态加载内容**：切换到手动抓取模式，等待加载后再点击扩展
- **受保护页面**：银行、支付等页面禁止扩展访问
- **特殊结构**：尝试切换提取模式或添加自定义选择器

### 提取内容不完整或错误？

1. 切换提取模式：选择器模式 → 阅读模式 → 混合模式
2. 配置自定义选择器：打开选项页面添加网站特定选择器
3. 使用浏览器开发者工具检查 HTML 结构

### 悬浮按钮不显示？

- 检查扩展设置中"显示悬浮按钮"是否开启
- 新标签页、扩展页面不显示
- 设置更改后刷新页面

## AI 功能问题

### AI 功能无法使用？

1. 检查 API Key 是否正确且有效
2. 确认有使用额度
3. 检查网络和代理设置

### AI 响应速度慢？

- 选择更快的模型（如 gpt-3.5-turbo）
- 简化提示词，避免过长输入
- 配置更快的 API 端点或本地模型

### API 调用失败？

- **401 Unauthorized**：API Key 无效或过期
- **429 Rate Limit**：请求频率过高，稍后重试
- **网络错误**：检查连接，尝试更换 API 端点

## 性能问题

### 扩展运行缓慢？

1. 降低日志级别：选项 → 开发者设置 → Error/Silent
2. 关闭不需要的功能（自动抓取、调试面板）
3. 定期清理 AI 聊天历史

### 页面加载变慢？

- 切换到手动抓取模式
- 简化自定义选择器，避免通配符

## 数据问题

### 如何备份数据？

- 选项页面 → 导出配置（JSON 文件）
- 登录 Google 账号启用 Chrome 同步
- 导出 AI 聊天历史

### 数据丢失怎么办？

- 从备份导入配置文件
- Chrome 同步：重新登录账号等待同步

## 隐私和安全

### MoeCopy AI 会收集数据吗？

不会。所有数据本地存储。

### API Key 安全吗？

API Key 本地加密存储，不会上传服务器，仅用于 AI 调用。建议使用专用 Key 并定期更换。

## 其他问题

### 如何参与开发？

```bash
git clone https://github.com/yusixian/moe-copy-ai.git
```

Fork 项目 → 提交 PR → 遵循代码规范，添加测试用例

### 商业使用限制？

使用 AGPL-3.0 许可证，允许商业使用、修改和分发，需保留版权声明。

### 支持哪些网站？

**完全支持**：新闻、电商、博客、论坛等

**部分支持**：SPA 单页应用（需手动抓取）、动态加载内容（需等待加载）

**不支持**：Chrome 内置页面（chrome://）、扩展程序页面、受保护页面（网银等）

### 如何提高提取准确性？

1. 选择合适模式：文章页面用阅读模式，列表页面用选择器模式，复杂布局用混合模式
2. 配置常用网站专用选择器
3. 明确 AI 提示词
