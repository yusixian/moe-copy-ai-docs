# MoeCopy AI 文档

欢迎来到 MoeCopy AI 的官方文档！这里提供了完整的使用指南、配置说明和开发者资源。

## 什么是 MoeCopy AI？

MoeCopy AI 是一款基于 Plasmo 框架开发的浏览器扩展，专门用于智能提取网页中的结构化数据。它集成了先进的 AI 技术，能够自动识别网页内容并提供智能摘要功能。

### 核心特性

- 🎯 **智能提取**：多层级选择器系统，准确识别文章标题、内容、作者、日期等信息
- 🤖 **AI 增强**：集成 OpenAI API，提供内容摘要、翻译等智能功能
- 🌐 **广泛兼容**：支持主流新闻网站、博客、文档站点等各类网页
- 📱 **移动友好**：完美适配桌面和移动端（Kiwi Browser）
- 🎨 **精美界面**：现代化设计，支持深浅色主题切换
- 🔧 **高度可定制**：支持自定义选择器、提示词模板等

## 开始使用

### 第一步：安装扩展

从 Chrome 网上应用店安装 MoeCopy AI：

[🔗 安装扩展](https://chromewebstore.google.com/detail/moe-copy-ai/dfmlcfckmfgabpgbaobgapdfmjiihnck)

### 第二步：体验功能

1. 访问任何文章页面
2. 点击扩展图标
3. 查看自动提取的内容
4. 尝试 AI 摘要功能

### 第三步：个性化配置

根据您的需求调整设置：
- 配置 AI API（可选）
- 设置自定义选择器
- 调整界面偏好

## 技术亮点

### 智能选择器系统

MoeCopy AI 使用多级回退的选择器系统：

```
文章元素 → 自定义选择器 → 默认选择器 → 段落集合 → Body 内容
```

### AI 集成

- 支持 OpenAI 及兼容 API
- 流式输出，实时显示结果
- 自定义提示词模板
- 使用量统计和历史记录

### 数据处理

- 三种提取模式：选择器、阅读、混合
- 智能内容清理
- 多格式导出：JSON、Markdown、文本
- 完整的元数据提取
